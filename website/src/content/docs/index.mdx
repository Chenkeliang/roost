## What is Roost?

Roost is an open-source macOS configuration backup and migration tool. You
**select** what to manage, Roost stores it in **your own private git repo**, and
on another Mac you **load** it back — with a visual manager for what's backed up.

<CardGrid stagger>
  <Card title="Your data stays yours">
    No server, no account, no telemetry. Your config lives in your private git
    repository.
  </Card>
  <Card title="Reversible by default">
    `load` previews changes and backs up existing files before overwriting.
  </Card>
  <Card title="Two complementary layers">
    Back up existing dotfiles as-is, and manage portable aliases / env / PATH
    that Roost regenerates per machine.
  </Card>
  <Card title="Built on trusted tools">
    Thin orchestration over chezmoi, Homebrew, age, and mise — never reimplementing them.
  </Card>
</CardGrid>
